# 🏍️ Motorbike Rental System

## 📌 Description
This is a Java web application that allows users to rent motorbikes online.  
The system supports booking, managing motorbikes, and handling rental transactions.

---

## 🛠 Tech Stack
- Java (Servlet, JSP)
- JDBC
- Microsoft SQL Server
- HTML, CSS, JavaScript

---

## 🚀 Features
- 🔐 User authentication (Login / Register)
- 🏍️ View and rent motorbikes
- 📦 Booking management
- 🖼️ Upload motorbike images
- 🛠️ Admin management (CRUD motorbikes)

---

## 📁 Project Structure
- `controller/` – Handle requests (Servlet)
- `dao/` – Database access
- `model/` – Data objects
- `view/` – JSP pages

---

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/hongdz0000/motobikerental.git

2. Open project in NetBeans or IntelliJ

3. Setup database in Microsoft SQL Server

4. Configure database connection in code

5. Run project on Apache Tomcat
